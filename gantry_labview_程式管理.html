Gantry System LabVIEW MAC

我們利用git來做LabVIEW程式的檔案管理，方便離線不使用Gantry PC時還是可以在自己的電腦繼續改code並正確的回傳到Gantry PC

Git repository path : admin@192.168.50.206:/volume1/git_repos

1. 使用說明

  • 打開 Git Bash
# 移動到程式的資料夾
cd d:
cd Gantry_General

# 每天做任何改動前先更新到最新版（可以考慮加進開LabVIEW流程）
git pull

# 確認今天做了什麼改動
git status

# 新增所有檔案（除了 *.aliases 檔）
git add --all && git reset -- *.aliases

# 確認所有改動檔案
git commit -a -m "update"

# 上傳所有改動檔案
git push


# 更改 ~/.bashrc，新增簡易指令
pull="git pull"
status="git status"
add="git add --all && git reset -- *.aliases"
commit="git commit -a -m \"update\""
push="git push"

2. 初始設定

當新的PC想要使用此功能請完成以下設定

  • 第一步請先 [下載 Git Bash](https://gitforwindows.org)，下載都用預設就可以
  • 打開Git Bash，會出現一個終端機(terminal)的畫面
  • 接下來的指令請在terminal上完成

2.1. 下載LabVIEW程式

cd <PATH-TO-CLONE>
git clone admin@192.168.50.206:/volume1/git_repos/<repo>
cd <repo>
git checkout dev

2.2. 免密碼設定

# 創立金鑰
ssh-keygen
scp ~/.ssh/id_dsa.pub admin@192.168.50.206:~/
password:

ssh admin@192.168.50.206
sudo cat id_dsa.pub >> ./.ssh/authorized_keys
exit

2.3. 一些在架設此功能遇到的問題

  • NAS未開通使用金鑰：請參考 此連結
    • 所有700, 600.. 請都改成755
  • 需要重新啟動NAS ssh功能：請參考 此連結
    • Synology App 安裝在923第一台電腦
